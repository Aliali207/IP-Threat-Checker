# 🛡️ IP Threat Intelligence Checker

### 📖 Description
A Python-based security tool designed for **SOC Analysts**. It automates the process of checking IP reputations against **90+ security vendors** using the **VirusTotal API v3**.

### 🚀 Features
- **Fast Analysis:** Instantly get malicious and suspicious counts.
- **Security First:** Implements environment variables to protect API keys.
- **Clean Output:** Provides clear verdicts (Warning/Clean) for better decision-making.

### 🛠️ Setup
1. **Clone the repo:**
   `git clone https://github.com/Aliali207/IP-Threat-Checker.git`
2. **Install dependencies:**
   `pip install requests python-dotenv`
3. **Configure Environment:**
   Create a `.env` file and add your key: `VT_API_KEY=your_key_here`
4. **Run the tool:**
   `python python-scripty.py`
